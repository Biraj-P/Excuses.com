API Configuration

Note: This page is for development and testing purposes. In production, API keys should be stored securely on the server.

Why do I need an API key?

The Excuses.com application uses the Together API to generate creative excuses using the Meta-Llama/Llama-3.3-70B model.

When running on platforms like GitHub Pages that don't support server-side code, you need to provide your own API key.

Get your API key from Together.ai

Security Note

Your API key is stored in your browser's local storage and is only sent directly to the Together API.

For better security, deploy the application to a platform that supports server-side code or serverless functions.

Cache Management

Excuses are cached locally to improve performance and reduce API calls.
